시사점 — 전 진영 공통 방향 (D-level)

① 모델 → 하네스 경쟁

2026년 차별화는 모델 점수보다 에이전트 운영 구조(하네스·플랫폼·표준)에서 발생. 빅3 모두 CLI·IDE·클라우드·브라우저로 에이전트 접점을 다층화.

② 표준 수렴

MCP(도구 연결)·AGENTS.md(지침)·Agent Skills(노하우)가 벤더 경계를 넘는 공통 규격으로 — 한 번 만든 자산이 도구를 갈아타도 살아남음. 도입 시 표준 준수 자산에 투자할 것.

③ 비개발자 확장

Cowork(Anthropic)·Atlas agent mode(OpenAI)·NotebookLM Deep Research(Google) — 코딩을 넘어 일반 지식노동 자동화로 전선 확대. 사무직 대상 교육 수요 증가.

④ 오픈웨이트 주도권 = 중국

Meta가 클로즈드로 선회한 사이 DeepSeek·Qwen·Kimi·GLM·MiniMax가 agentic 특화 오픈웨이트를 MIT/Apache로 공급 — 자체 호스팅·커스텀의 사실상 유일한 프런티어급 선택지. 단 데이터 거버넌스 검토는 별도.

⑤ "오픈"은 스펙트럼

같은 "오픈 모델"이라도 Apache/MIT ↔ Modified MIT ↔ 벤더 라이선스 ↔ 700M MAU 조항까지 조건이 제각각 — 도입 전 라이선스 원문을 모델 단위로 확인할 것 (위 표 참조).

⑥ 추론 비용이 아키텍처를 결정

sparse MoE(활성 3–10%)·Mamba 하이브리드(Nemotron 3)·Sparse Attention(DSA)·MTP — 전 진영이 에이전트의 토큰 대량 소비를 겨냥한 추론 효율 기술로 수렴. 장시간 자율 루프의 비용 구조가 모델 설계를 끌고 가는 중.
