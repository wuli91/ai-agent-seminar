사전 준비 가이드

첫 시간부터 직접 실습할 수 있도록, 세미나 전에 아래 3가지를 준비해 주세요. 어려우면 진행자에게 미리 도움을 요청하세요 — 막히는 게 정상입니다. 설치·로그인이 안 되면 트러블슈팅 가이드를 먼저 확인하세요.

① 계정

Claude 계정 생성 & 로그인 (1회차 전)

② 노트북

실습용 노트북 + 인터넷 (관리자 권한이면 더 좋음)

③ Claude Code

터미널 도구 설치 (3회차 전까지)

⚠ 가장 먼저: 데이터 보안 원칙 회사 정책상 외부 AI에 넣어도 되는 정보 / 넣으면 안 되는 정보를 먼저 확인하세요. 고객 개인정보, 미공개 재무, 계약·법무 비밀, 보안 키 등은 넣지 않는 것이 기본입니다. 자세한 기준은 1회차에서 다룹니다. (회사 보안팀 가이드를 우선합니다.)

1. Claude 계정 만들기 (Claude 앱)

Claude 앱은 브라우저(claude.ai) 또는 데스크톱 앱으로 씁니다. 별도 설치 없이 브라우저만으로 충분합니다.

  • 브라우저에서 claude.ai 접속
  • 회사 이메일(또는 진행자가 안내한 계정)으로 로그인 / 가입
  • 로그인 후 채팅 화면이 보이면 준비 완료
  • (선택) 데스크톱 앱 설치: claude.ai 하단 "Download" → macOS/Windows 선택
플랜 안내 — 회사 계정을 쓰세요 실무 도입은 보안상 Team 플랜 이상을 권장하며, 회사가 미리 발급한 계정(워크스페이스)으로 로그인하시면 됩니다. 초대 메일이 안 왔거나 헷갈리면 도입 담당/진행자에게 문의하세요. 개인 무료 플랜은 맛보기엔 가능하지만 민감한 업무 데이터에는 사용하지 마세요. (플랜·보안 차이: 요금제 & 보안 비교)

2. 노트북 준비

항목권장
OSmacOS 12+ 또는 Windows 10/11
브라우저Chrome 또는 Edge 최신 버전
권한소프트웨어 설치 가능(관리자) — 막히면 IT에 사전 요청
네트워크사내/외부 인터넷 접속 (방화벽으로 막히면 진행자에게 문의)

3. Claude Code 설치 (3회차 전까지)

Claude Code는 터미널에서 실행하는 도구입니다. 터미널이 처음이라도 괜찮습니다 — 아래 명령을 그대로 복사해 붙여넣으면 됩니다. 3회차에서 함께 설치를 점검하니, 미리 해보고 안 되면 그대로 오셔도 됩니다.

🍎 macOS

터미널 열기: ⌘ + Space → "터미널" 입력 → Enter

# 1) 설치 (한 줄 복사·붙여넣기)
curl -fsSL https://claude.com/install.sh | bash

# 2) 새 터미널 창을 열고 확인
claude --version

# 3) 로그인 (브라우저가 열립니다)
claude

🪟 Windows

PowerShell 열기: 시작 → "PowerShell" 검색 → 실행

# 1) 설치 (한 줄 복사·붙여넣기)
irm https://claude.com/install.ps1 | iex

# 2) 새 창을 열고 확인
claude --version

# 3) 로그인 (브라우저가 열립니다)
claude
설치가 안 될 때 ① 명령을 한 줄 통째로 복사했는지 확인 → ② 터미널/PowerShell을 새로 열어 다시 시도 → ③ 회사 노트북은 보안 정책으로 막힐 수 있으니 IT/진행자에게 문의하세요. 3회차에 함께 해결합니다.

최종 체크리스트

  • 회사 데이터 보안 기준을 확인했다 (무엇을 넣어도 되는지)
  • claude.ai 에 로그인된다
  • 노트북에 소프트웨어를 설치할 수 있다(또는 IT에 요청해 두었다)
  • 터미널/PowerShell에서 claude --version 이 동작한다 (3회차 전까지)
  • "내 업무 중 반복되는 일" 1~2가지를 적어왔다
