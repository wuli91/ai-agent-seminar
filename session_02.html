'Cowork' = AI를 동료처럼 쓰기

맥락을 기억

프로젝트에 자료를 넣어두면 매번 설명할 필요 없음.

자료를 읽음

보고서·PDF·표를 올리면 그 내용을 바탕으로 작업.

결과물을 만듦

Artifacts로 문서·표·슬라이드 초안·간단한 도구까지.

"ChatGPT·Gemini·웍스AI랑 뭐가 달라요?" → 개념은 같습니다 — AI 앱 기능 지도 (기능별 의도·쓰임·화면 위치 + 4개 앱 매핑)

Projects — 내 업무 전용 작업공간

하나의 주제(업무)에 대한 대화·자료·지침을 한곳에 모읍니다.

보안프로젝트에 올리는 자료도 Session 1 기준 적용 — 기밀은 익명화하거나 올리지 않습니다.

Artifacts — 대화 옆에 '결과물'이 만들어진다

이런 걸 만든다

· 보고서·기획서·공지문 문서
· 비교표·체크리스트
· 간단한 계산기·설문 등 작은 도구
· 슬라이드 초안 텍스트

좋은 점

· 바로 보면서 "여기 고쳐" 가능
· 버전이 옆에 쌓임
· 복사/다운로드해서 바로 사용

사용법"~를 문서로 만들어줘" / "~를 표로 정리해줘"라고 하면 오른쪽에 결과물이 생깁니다.

직무별 — 오늘 당장 되는 일

직무이렇게 시킨다
기획/PM"이 자료로 1페이지 기획 초안 + 리스크 3가지"
마케팅/영업"이 제품 설명으로 광고 카피 5종 + 고객 반론 대응"
디자인/콘텐츠"이 글을 SNS 3편으로, 톤은 친근하게"
경영지원/HR"이 규정 문서를 직원 FAQ 10개로"
재무/데이터"이 표를 읽고 추세·이상치·요약 코멘트"

자세한 시나리오·프롬프트는 직무별 핸드아웃에 있습니다.

데모 ① 긴 문서 → 임원 요약

(보고서 PDF 업로드 후)

이 문서를 임원 보고용으로 정리해줘.
- 핵심 결론 3가지 (각 한 줄)
- 의사결정이 필요한 항목 표 (이슈/선택지/추천)
- 리스크와 다음 단계
- 전체 한 페이지 이내

20장 보고서를 한 페이지 의사결정 문서로. 단, 숫자는 원문과 대조 확인.

데모 ② 빈 화면 공포 없애기 — 초안

너는 우리 회사 기획자야. (프로젝트에 회사 배경 저장됨)

신규 기능 '스마트 알림'의 내부 제안서 초안을 써줘.
- 배경 / 문제 / 제안 / 기대효과 / 예상 리스크
- 표와 불릿 활용, 정중한 보고체
- 빠진 정보는 [확인 필요] 로 표시
핵심AI는 0→70%를 빠르게. 나머지 30%(판단·사실확인)는 사람이 채웁니다.

🙌 핸즈온 — 내 직무 프로젝트 만들기 Claude 앱

  1. 새 Project 생성, 이름은 내 업무로 (예: "마케팅 카피 작업실")
  2. 프로젝트 지침에 배경·톤·형식을 적어 저장 — 실습 소재 모음의 지침 템플릿을 채우면 됩니다
  3. 참고 자료 1개 업로드 (익명화된 예시/공개 자료 — 없으면 실습 소재 모음의 샘플 회의록)
  4. 실제 업무 1건을 요청해 Artifact 결과물 1개 만들기
  5. "더 짧게/표로/톤 변경"으로 2~3번 다듬기
기밀 주의업로드 전에 "이 자료 넣어도 되나?" 한 번 더 확인.

잘 쓰는 사람들의 습관

지침을 재사용

잘 나온 프롬프트는 프로젝트 지침/메모에 저장해 재사용.

예시를 준다

"이런 느낌으로"라며 좋은 샘플 1개를 붙여넣으면 품질 급상승.

나눠서 시킨다

큰 일은 단계로 쪼개 요청 → 중간 점검.

항상 검토

사실·숫자·고유명사는 원본 대조. 책임은 사람에게.

앱만으로는 아쉬운 순간

파일이 많을 때

폴더 안 파일 50개를 일괄 처리? 하나씩 업로드는 비효율.

반복 작업

매주 같은 정리·변환을 손으로 반복.

→ 그래서 다음 세션 Claude Code: 내 컴퓨터의 폴더·파일을 직접 다루고 자동화합니다.
